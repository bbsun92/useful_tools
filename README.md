# useful_tools
Miscellaneous collection of useful tools

- ***make_dir.py***:
Creates a structured project folder with the structure, can be adjusted as required:
```bash
├── admin
│   ├── drafts
│   ├── external
│   ├── formal
│   └── internal
├── analysis
│   ├── input
│   ├── interim
│   ├── output
│   └── scripts
└── dissemination
    ├── presentations
    └── reports
```
